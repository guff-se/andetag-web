# ANDETAG

MALIN & GUSTAF TADAA

## Headings

Use Jost (Google Font) for names and headings, in UPPERCASE, with 0.3em letter spacing. The brand name ANDETAG shouold *always* be written this way.

Jost:
https://fonts.google.com/specimen/Jost

## Body Text

Use Baskervville for body text.
https://fonts.google.com/specimen/Baskervville


## Colors

- Background: light pink `#f7dcea` or black
- Other colors: aubergine `#4a0d2f` and lavender `#d0a4cc`
- Accent color: yellow-green `#e0e31c` (must not be used on black)
- Buttons:
  - Red `#bc2026`
  - Green `#5ebf84`
