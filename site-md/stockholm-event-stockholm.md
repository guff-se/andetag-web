# Event Stockholm – ett annat slags evenemang

Event i Stockholm finns i många former.

ANDETAG erbjuder något annat: ett stillsamt rum för [grupper](https://www.andetag.museum/stockholm/gruppbokning/) och [företag](https://www.andetag.museum/stockholm/foretagsevent/).

Inga högtalare.

Ingen networking-musik.

Istället ljus, textil och ett gemensamt andetag.

Vi kan ta emot mindre grupper för [privata visningar](https://www.andetag.museum/stockholm/gruppbokning/). Morgon, dag, kväll eller helg.  
Ett event anpassat efter er – inte tvärtom.

[ Boka biljetter nu! ](https://www.andetag.museum/stockholm/biljetter/)

## Events Stockholm – när rummet är ert

En vanlig besökare går igenom utställningen i sin egen takt, tillsammans med andra gäster.

Vid ett privat event har gruppen rummet för sig själv.

  


Passar för:

– Företag som söker ett lugnt alternativ till konferens eller middag

– Mindre grupper som vill uppleva konst tillsammans

– Särskilda tillfällen – födelsedagar, avsked, gemensamma pauser

  


Eventet kan kombineras med enkel catering.

Vi hjälper gärna till med vad som passar er.

## Ett event utan spektakel

ANDETAG är inte ett event i bemärkelsen fest eller show.

Det är ett rum där gruppen får vara tillsammans på ett annat sätt.

Stillhet.

Närvaro.

Andning.

  


Ljuset och musiken tar hand om miljön. Ni tar hand om varandra.

## Kontakt för event

Skicka ett mail till [info@tadaa.se](mailto:info@tadaa.se) med önskat datum, antal deltagare och eventuella önskemål.

Vi återkommer med förslag och pris.
